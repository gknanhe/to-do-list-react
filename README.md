# ToDo App

this is simple todo list app where user can create , update, delete, and mark as done to todods.

## Usage

1. to add todo click on + icon 
2. to edit and delete click on 3 dots.

## Installation

fork the repo or download the code.
in vs code run

    ``` npm install

    ``` npm run dev
